# AI Personalized Career Guidance System

## Technologies Used

- Python 3.12
- Django
- Bootstrap
- HTML
- CSS
- JavaScript
- Scikit-learn
- Pandas
- NumPy

## Installation

### Clone Repository

```bash
git clone https://github.com/YourUsername/AI-Personalized-Career-Guidance.git
```

### Go inside project

```bash
cd AI-Personalized-Career-Guidance
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run migrations

```bash
python manage.py migrate
```

### Start server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000
```

## Features

- Student Registration
- Student Login
- AI Career Assessment
- Machine Learning Prediction
- Skills Recommendation
- Course Recommendation
- Career Dashboard
